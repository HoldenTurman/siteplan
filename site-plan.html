My Projects

Restaurant Reservation App This app allows restaurant managers to view what reservations are available at certain times and dates while allowing the user to make reservations by taking consumer information such as name, party size, date, and phone number. It also lets them know what tables are currently reserved/not available. Languages used: JavaScript, nodeJS, React, express, PostgreSQL, knex, HTML, Bootstrap

ER.Portfolio This is a portfolio for a client's personal photography website. It allows the user to showcase their photos taken while also sharing more info allowing a consumer to get to know the photographer more. Languages used: HTML, CSS, TailwindCSS

Airbnb Website This is a mock-up of an Airbnb website. It has search capabilities as well as different types of drop-down usage. Displaying Images of the Airbnb in a nice and organized way. Languages used: HTML, CSS

Recipe Tracking App This is a recipe tracker. It allows a user to put in their favorite recipes and save them so they can access them easily whenever needed. Languages used: HTML, CSS, React, Bootstrap

Calculator App For this calculator app, it will function like any other calculator. What I did instead of incorporating a library to handle the math on its own. I decided to write. out the mathematics for it to make sure I had a better understanding of what was going on. Languages used: JavaScript, HTML, CSS
